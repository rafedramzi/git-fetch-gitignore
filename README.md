# Git Fetch-Ingore

Usage:

```
git fetch-ignore clean all|[repository...]
git fetch-ignore sync git-repo
git fetch-ignore sync git-repo [OPTION] [repository...]

git fetch-ignore get [lang...]
git fetch-ignore get [OPTION] [path...]
```

Available Source:

- git repository
- url
- gitignore.io
